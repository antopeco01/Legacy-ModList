The Minecraft Legacy Modlist (1.7.10)

    This list was created to provide all modded Minecraft players, both new and experienced, with a complete and exhaustive resource for all the material available to create a modpack for Minecraft Legacy 1.7.10. The goal is to offer a single place where anyone interested in creating a modpack for this version can find all the information they need, without having to search around and piece it together like the creator of this list had to.


    The list is divided into categories and is easy to navigate thanks to the Navbar in the top right corner, for each category there is a table showing the mods that fall under it, divided into four columns:
    Name, Info ( a short description to give an idea of what the mod does ), Addons ( mods that add content and rely on the main mod ), and Fork ( unofficial releases, along with a description of how it differs from the official one. ).
    Clicking on the name of a mod will display a tooltip with the links to the latest versions for download, and hovering over a category in the Navbar menu will show one with a brief description of the types of mod that can be found under that category, at the bottom of the table there's also a section summarizing the contents of the category.


    Many mods require the installation of other mods to function properly, these are called Dependencies. There are several ways to find them:

    • If the mod is on CurseForge, go to the Relations section ( if available ), then filter by Required Dependencies. Often, dependencies are also listed in the Description section by the creator.
    • If the mod is on Modrinth, go to the Versions section, select the desired version, and scroll down to the Dependencies section to see those marked as required.
    • If the mod is hosted on GitHub or another forum, you might find information about its dependencies directly there. If not, a quick search is usually enough to find out.


    As for forks, in most cases they are basically updated and maintained versions of mods whose support from the original creator has ceased, they often feature improved performance, bug fixes, and updated build scripts, however this is not always the case, as some may not retain the original mod's status or license. Most are maintained by the GTNH  team, who adapt them for their own modpack, which may involve changes to recipes and mechanics, consult their repositories and refer to this list for more informations.
    Note :  Forks of mods that depend on library mods or APIs often require forked versions of those libraries as well ( e.g., NEI - GTNH requires CodeChickenCore - GTNH ). If you choose to use forks, keep in mind that compatibility is not always guaranteed.


    All the material found here is the result of thorough research and comes from many reliable sources, including major modded Minecraft forums, GitHub profiles of mod developers, articles and lists created by the community, and a detailed analysis of the most popular and widely played modpacks for this version, including those released at launch as well as those introduced in the years since and still actively maintained today. Please keep in mind that, despite the many hours spent creating this list, some mods, even quite well known ones, might be missing. The goal is to keep the list updated with new releases and community suggestions, while maintaining the same level of detail and information to ensure it remains as usable as possible.
